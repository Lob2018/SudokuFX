Detailed Description

Component providing conversions between different Sudoku grid representations:

Interface defining methods to convert between different representations of a Sudoku grid.

  • gridValue : player's input as a comma-separated String of 81 values
  • defaultGridValue : default grid as an 81-character String
  • int[] : numerical representation of the grid
  • List<String> : list of 81 cell values as Strings; for gridValue, each cell may be:
    • a combination of digits 1-9 (no repetitions within the cell)
    • or "0" / empty string if the cell is empty

Validation ensures that each cell in listToGridValue(List) contains only valid digits and no repeated numbers. Empty or blank cells are normalized to "0". Null arguments will throw NullPointerException.
