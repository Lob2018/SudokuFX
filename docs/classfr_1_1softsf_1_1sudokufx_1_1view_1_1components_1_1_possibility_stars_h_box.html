Detailed Description

A custom HBox that displays star ratings based on a percentage value.

The star ratings are represented by Unicode characters:

  • Full star: \ue838
  • Half star: \ue839
  • Empty star: \ue83a This component also provides formatted accessibility and tooltip text bindings based on the current percentage and visibility.
