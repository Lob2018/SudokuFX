Executes a long-running file task asynchronously with UI notifications.

The task runs on a background thread. A spinner and toast are shown during processing. On success, the result is passed to onSuccess. Errors are handled internally (logged, toast shown, spinner hidden); no error callback is needed.

Parameters
<T>the result type
selectedFilethe file to process; must not be null
spinnerthe UI spinner to indicate progress; must not be null
toasterthe UI toaster to display messages; must not be null
taskFunctionfunction to process the file and return a result; must not be null
onSuccesscallback executed on the UI thread upon success; must not be null
Exceptions
NullPointerExceptionif any required parameter is null
