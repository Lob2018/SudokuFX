Detailed Description

Utility class providing custom mapping methods for use with MapStruct.

These methods are used to simplify object mapping, particularly to avoid full graph traversal and handle reference cycles or partial entity instantiations.

Selects and returns the first Game from the given set, or null if the set is null or empty.

Used by MapStruct to simplify mapping of Player → PlayerDto by avoiding full game set mapping. This helps prevent circular references and reduces DTO complexity.

Note: The order is not guaranteed; selection is non-deterministic.

Parameters
gamesthe set of Game entities, may be null or empty
Returns
one Game from the set, or null if none
