Detailed Description

ViewModel for managing and applying background (color and image) to a GridPane.

  • Sets and applies background color. - Handles image file selection, validation, and asynchronous loading. - Resizes images to fit the available space while maintaining aspect ratio. - Displays notifications and a loading spinner during image loading. - Handles success and error cases during image loading.
