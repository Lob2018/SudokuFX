Detailed Description

A custom VBox component for displaying toast notifications.

This class extends JavaFX's VBox and provides methods to add and manage toast messages.

◆ temporizeToast()

void fr.softsf.sudokufx.view.component.toaster.ToasterVBox.temporizeToast ( final ToasterButton toast,
final double duration )
private

Sets a timer to automatically remove the toast after a specified duration.

This method ensures that the currently focused node is preserved when the toast is removed, allowing for a seamless user experience. Additionally, it guarantees that the narrator continues to read the toast message without interruption, providing accessibility for users who rely on screen readers.

Parameters
toastThe toast button to be removed.
durationThe duration in milliseconds after which the toast should be removed. The toast will be removed without disrupting the focus of the currently active node.
