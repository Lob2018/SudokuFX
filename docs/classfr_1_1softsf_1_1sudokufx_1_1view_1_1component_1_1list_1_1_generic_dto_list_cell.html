Detailed Description

A reusable ListCell for displaying items of type T in a ListView.

Each cell shows a label with text generated by a Function and a remove button. Accessible texts, tooltips, and confirmation messages are dynamically retrieved via java.util.function.Supplier. When the remove button is pressed, a confirmation fr.softsf.sudokufx.view.component.MyAlert is created and shown, allowing the user to confirm the removal of the item.

The cell clears its content when empty to avoid memory leaks and ensure proper reuse.

Parameters
<T>the type of items displayed in the ListView
