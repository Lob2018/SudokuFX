Detailed Description

A generic ListCell for displaying and removing items of type T in a
ListView.

Displays a label with text obtained from a provided function, and a button that removes the item after user confirmation via a localized dialog. Accessibility and tooltips are dynamically updated using StringBindings.

Parameters
<T>the type of items displayed in the ListView
