Detailed Description

Manages file system operations related to the application's data folder.

Defines file system operations related to the application's data management.

Implements IFileSystem to recursively delete the application's data directory, ensuring path validation and detailed logging. Throws IllegalArgumentException via ExceptionTools if input paths are null.
