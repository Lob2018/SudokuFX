Detailed Description

A custom JavaFX GridPane that displays a two-phase animated spinner using Unicode icons.

The spinner consists of two overlapping Text nodes with pulsing animations, creating a visual loading indicator. By default, the spinner is hidden and not managed by the layout. It can be shown or hidden using the showSpinner(boolean) method.
