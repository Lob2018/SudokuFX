Detailed Description

A custom JavaFX GridPane that displays a two-phase animated spinner using Unicode icons.

The spinner visibility is reactive and managed via a BooleanProperty, typically bound to a ViewModel to handle multiple concurrent asynchronous calls and features a smooth fade-out transition.
