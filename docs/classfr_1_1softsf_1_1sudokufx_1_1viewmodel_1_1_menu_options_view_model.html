Detailed Description

ViewModel managing menu options UI state and accessibility strings.

Provides localized StringBindings for button labels, tooltips, and roles, updating automatically on locale changes. Supports options color and image initialization and updates, including asynchronous image loading with user feedback.

◆ loadBackgroundImage()

void fr.softsf.sudokufx.viewmodel.MenuOptionsViewModel.loadBackgroundImage ( File selectedFile,
SpinnerGridPane spinner,
GridPane sudokuFX )

Loads a background image asynchronously and applies it to the specified GridPane.

The method validates that the file is non-null, exists, and is a valid image. If valid, it delegates asynchronous loading, resizing, and conversion to AsyncFileProcessorService. On success, the resulting BackgroundImage is applied to the provided GridPane. Errors are logged and shown as a toast via ToasterVBox. Saving the image path to the database should be handled externally.

Asynchronous processing ensures the UI thread is not blocked during image loading and resizing.

Parameters
selectedFilethe image file to load; must not be null, must exist, and be a valid image
spinnerthe spinner component to indicate loading; must not be null
sudokuFXthe GridPane where the background image will be applied; must not be null

void fr.softsf.sudokufx.viewmodel.MenuOptionsViewModel.updateSongPath ( String absolutePath,
String name )
private

Updates the audio file path (songPath) in the current player's options.

A copy of the current OptionsDto is created with the new path and persisted via OptionsService#updateOptions(OptionsDto). On success, the current player is refreshed, songProperty is updated, and an info toast is shown. On failure, an error toast is shown.

Parameters
absolutePaththe new audio file absolute path
namethe file name, used in toast messages
