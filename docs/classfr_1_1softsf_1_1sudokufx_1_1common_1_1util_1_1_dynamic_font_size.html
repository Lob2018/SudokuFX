Detailed Description

Dynamically adjusts the root font size of a JavaFX Scene based on its dimensions.

Uses a debounced dynamic stylesheet to minimize memory allocations and reduce GC pressure. Updates occur only when the font size changes significantly, leveraging JavaFX's stylesheet caching.

◆ updateFontSize()

void fr.softsf.sudokufx.common.util.DynamicFontSize.updateFontSize ( )

Updates the font size of the scene's root node based on its current dimensions.

The font size scales with the smaller dimension of the scene using FONT_SIZE_RATIO. Updates are applied only if the size changes significantly, reducing unnecessary stylesheet refreshes and memory allocations. A dynamic stylesheet is created or updated in the scene's stylesheets, leveraging JavaFX's internal caching.
