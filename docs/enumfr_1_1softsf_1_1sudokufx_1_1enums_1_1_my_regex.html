Detailed Description

Utility class for managing and validating text against precompiled regular expressions.

This class includes methods for password validation, semantic versioning, alphanumeric string validation, and flexible contextual validations with error handling.

The class is implemented as a Singleton using an enum to ensure a single shared instance. It uses the SLF4J logging framework for error reporting.

boolean fr.softsf.sudokufx.enums.MyRegex.isValidatedByRegex ( final String text,
final Pattern pattern )

Validates the given text against the specified regular expression pattern.

If the provided pattern is equal to the internal secretPattern, this method performs a stricter password validation by delegating to isValidPassword(String). Otherwise, it performs a generic regex match using the provided pattern.

Parameters
textThe text to validate; must not be null.
patternThe regular expression pattern to validate against; must not be null.
Returns
true if the text matches the regex pattern or meets the password criteria; false otherwise.
Exceptions
NullPointerExceptionif text or pattern is null.

◆ isValidPassword()

boolean fr.softsf.sudokufx.enums.MyRegex.isValidPassword ( final String password)
private

Validates a secure password against strict security criteria:

  • Exactly 24 characters in length.
  • Contains only letters (uppercase and lowercase), digits, and allowed special characters @#$%^&()!.
  • At least 2 lowercase letters.
  • At least 2 uppercase letters.
  • At least 2 digits.
  • At least 2 special characters from @#$%^&()!.
Parameters
passwordThe password string to validate; must not be null.
Returns
true if the password meets all the above criteria; false otherwise.
Exceptions
NullPointerExceptionif password is null.
