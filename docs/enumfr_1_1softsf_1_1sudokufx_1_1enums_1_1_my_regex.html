Detailed Description

Utility class for managing and validating text against precompiled regular expressions.

This class includes methods for password validation, semantic versioning, alphanumeric string validation, and flexible contextual validations with error handling.

The class is implemented as a Singleton using an enum to ensure a single shared instance. It uses the SLF4J logging framework for error reporting.

@NotNull final Pattern pattern )

Validates the given text against the specified regular expression pattern.

For stricter cases like password validation, the method delegates validation to isValidPassword. If the pattern matches the secretPattern, this method performs stricter password validation with additional rules. Otherwise, it uses the provided pattern for generic regex validation.

Parameters
textThe text to validate (must not be null).
patternThe regular expression pattern for validation (must not be null).
Returns
true if the text matches the regex pattern; false otherwise.

boolean fr.softsf.sudokufx.enums.MyRegex.isValidPassword ( @NotNull final String password)
private

Validates a secure password based on strict requirements:

  • At least 2 lowercase characters.
  • At least 2 uppercase characters.
  • At least 2 numeric digits.
  • At least 2 special characters from #$%^&()!.
  • Exactly 24 characters in total.
Parameters
passwordThe password to validate (must not be null).
Returns
true if the password meets all security criteria; false otherwise.
