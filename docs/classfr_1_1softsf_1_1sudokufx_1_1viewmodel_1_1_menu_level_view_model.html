Detailed Description

ViewModel for menu difficulty levels, managing the selected difficulty level state and related UI bindings such as labels, accessibility texts, and visibility.

Provides reactive properties and bindings for JavaFX views, encapsulating business logic for difficulty selection.

This class is a Spring component and intended for dependency injection.

Note: implementation is not final and may evolve.
