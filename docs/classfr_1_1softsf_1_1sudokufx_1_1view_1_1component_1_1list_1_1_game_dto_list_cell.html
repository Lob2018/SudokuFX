Detailed Description

A custom ListCell for displaying a GameDto in a ListView, with a label showing the formatted update date and a button to remove the item after user confirmation.

The removal button and confirmation dialog texts are fully internationalized using
StringBinding. Accessibility and tooltip texts are updated dynamically based on the current item.
