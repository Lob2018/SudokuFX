Detailed Description

Service for managing PlayerLanguage entities.

Provides retrieval operations for player languages and mapping to PlayerLanguageDto. Validates DTOs using JakartaValidator after mapping. Intended for use by higher-level components such as coordinators or controllers to obtain language data.

Throws IllegalArgumentException if a requested language is not found in the repository.
