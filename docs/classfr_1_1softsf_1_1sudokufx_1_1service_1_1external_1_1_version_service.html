Detailed Description

Service for checking if the application version is up to date by querying the GitHub API.

It retrieves the latest release tag from the repository and compares it with the current application version. This service manages UI feedback through a spinner and status messages.

All method parameters and return values in this package are non-null by default, thanks to the @NonNullApi annotation at the package level.

◆ checkLatestVersion()

Task< Boolean > fr.softsf.sudokufx.service.external.VersionService.checkLatestVersion ( )

Checks if the current application version matches the latest release on GitHub.

Runs asynchronously using a JavaFX Task. It manages the loading spinner through the task's lifecycle and updates status messages for the UI. In case of network errors or timeouts, it defaults to assuming the version is up-to-date to avoid blocking the user.

Returns
a Task returning true if the version is up-to-date or if an error occurred, false if a new update is available

boolean fr.softsf.sudokufx.service.external.VersionService.parseResponse ( String json)
private

Parses the GitHub API JSON response to identify the latest version tag.

The method extracts the first tag name, normalizes it by removing the 'v' prefix, and performs a strict equality check against the current version.

Parameters
jsonthe JSON response string from GitHub API
Returns
true if versions match or if parsing fails; false if a different version is detected
Exceptions
IllegalArgumentExceptionif the JSON input is blank
