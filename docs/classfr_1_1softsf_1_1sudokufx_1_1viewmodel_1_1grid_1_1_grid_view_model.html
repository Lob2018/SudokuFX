Detailed Description

ViewModel responsible for managing a 9x9 Sudoku grid.

Provides observable cell view models, persistence hooks, and methods for applying difficulty levels, validating completion, and handling game state updates.

◆ getCurrentGridFromModel()

Optional< CurrentGrid > fr.softsf.sudokufx.viewmodel.grid.GridViewModel.getCurrentGridFromModel ( )

Loads the current Sudoku grid from the model into the ViewModel.

This method performs the following steps:

  1. Checks if a default grid exists for the selected game; returns empty if not.
  2. Reads the difficulty level and completion percentage directly from the model.
  3. Loads the default grid values into the ViewModel (with editable or not).
  4. Loads the current grid values into the ViewModel.
Returns
an Optional containing a CurrentGrid with the difficulty level and completion percentage if a grid exists, otherwise Optional#empty().
Exceptions
NullPointerExceptionif the selected game or its grid data is null

◆ getCurrentSolvedGridFromModel()

Optional< CurrentGrid > fr.softsf.sudokufx.viewmodel.grid.GridViewModel.getCurrentSolvedGridFromModel ( )

Loads the solved Sudoku grid from the model into the ViewModel.

This method performs the following steps:

  1. Checks if a default grid exists for the selected game; returns empty if not.
  2. Solves the current player grid if solvable; otherwise falls back to solving the default grid.
  3. Reads the difficulty level directly from the model.
  4. Loads the default grid values into the ViewModel (editable).
  5. Loads the solved grid values into the ViewModel (non-editable).
Returns
an Optional containing a CurrentGrid with the difficulty level and a completion percentage of 0 if a grid exists, otherwise Optional#empty().
Exceptions
NullPointerExceptionif the selected game or its grid data is null

◆ init()

void fr.softsf.sudokufx.viewmodel.grid.GridViewModel.init ( )

Initializes the 9x9 Sudoku grid by creating 81 GridCellViewModel instances with unique IDs and row/column coordinates.

Each cell's text property is attached to a listener that calls handleCellTextChange() whenever the text changes.

Must be called before accessing or manipulating grid cells; otherwise, IllegalStateException may be thrown.

void fr.softsf.sudokufx.viewmodel.grid.GridViewModel.setValues ( List< String > values,
boolean checkEditable )

Sets all 81 cell values and optionally their editability.

  • When checkEditable is true, cells with value "0" are editable, others are not.
  • When checkEditable is false, editability is left unchanged (used for player-entered grids).
  • Listeners are suppressed for all cells during bulk updates.
Parameters

checkEditablewhether to update cell editability
Exceptions
