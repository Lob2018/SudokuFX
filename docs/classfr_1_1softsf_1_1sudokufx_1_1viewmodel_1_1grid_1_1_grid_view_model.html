Detailed Description

ViewModel component managing a 9x9 Sudoku grid.

Provides observable cell view models and methods to manipulate grid data.

◆ init()

void fr.softsf.sudokufx.viewmodel.grid.GridViewModel.init ( ToasterVBox toaster)

Initializes the 9x9 Sudoku grid by creating 81 GridCellViewModel instances, each with a unique ID and coordinates.

Also sets up listeners on each cell's text property to:

  • Automatically verify the grid when in SOLVE mode or when all cells are completed with one value.
  • Stop any playing audio otherwise.

This method should be called once after bean construction.

Parameters
toasterthe ToasterVBox instance for displaying notifications if needed
