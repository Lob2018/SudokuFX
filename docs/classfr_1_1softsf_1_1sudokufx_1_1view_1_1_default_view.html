◆ initializeDifficultyButton()

void fr.softsf.sudokufx.view.DefaultView.initializeDifficultyButton ( DifficultyLevel difficultyLevel,
PossibilityStarsHBox possibilityStarsBox,
Button difficultyButton,
Label difficultyLabel,
String accessibilityKey,
String labelKey,
Button miniDifficultyButton )
private

Configures UI bindings for a difficulty-level button with full accessibility and dynamic visual feedback.

Supports screen readers and updates visibility, tooltips, labels, role description, and styling based on the selected level and percentage.

  • Visibility binding: The HBox is visible only when the difficulty level matches the selected one.
  • Accessible role description binding: Sets a localized description (e.g., "selected") on both buttons for screen readers when active.
  • Accessibility text binding: Binds a computed, localized text including the difficulty level and percentage for screen readers.
  • Tooltip binding: Tooltip text for both buttons is dynamically updated based on the accessibility text.
  • Button styling: Applies a pseudo-class to visually indicate which difficulty level is selected.
Parameters
difficultyLevelthe difficulty level to configure
possibilityStarsBoxthe HBox containing the possibility stars for the difficulty level
difficultyButtonthe main button for the difficulty level
difficultyLabelthe label displaying the difficulty level's name
accessibilityKeythe I18n key for the accessibility text
labelKeythe I18n key for the button's localized label text
miniDifficultyButtonthe mini button associated with the difficulty level
