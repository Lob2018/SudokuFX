void fr.softsf.sudokufx.view.DefaultView.setMenuHBoxPossibilitiesFromPercentage ( HBox starsContainer,
int percentage )
private

Updates the star ratings displayed in the provided HBox container based on a percentage value.

The star ratings are represented by Unicode characters:

  • Full star: \ue838
  • Half star: \ue839
  • Empty star: \ue83a
Parameters
starsContainerThe HBox container holding Text nodes representing stars.
percentageThe percentage value used to determine the star rating (from 0 to 100).
