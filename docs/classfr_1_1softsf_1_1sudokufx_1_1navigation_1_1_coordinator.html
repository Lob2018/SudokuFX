Detailed Description

Coordinator is a Spring-managed component that handles navigation and UI logic in a JavaFX application following the MVVM-C architecture.

Responsibilities include FXML view loading, scene switching, dynamic font scaling, and language toggling.

public< T > T fr.softsf.sudokufx.navigation.Coordinator.setRootByFXMLName ( final String fxml)
package

Loads the given FXML view, sets it as the root of the default scene, and returns its controller.

Resets the FXMLLoader, loads the view from resources, updates the scene root, and adjusts the font size if applicable.

On failure, logs the error, exits the app, and returns null.

Parameters
<T>the controller type
fxmlthe FXML base name (without extension)
Returns
the controller instance, or null if loading fails
